    Aplicación desarrollada con **Expo + React Native**.

    El proyecto **MicraCity** está construido con un conjunto de herramientas modernas del ecosistema **Expo y React Native**.

    Estas librerías se utilizan para gestionar:

    - Navegación
    - Estado global
    - Comunicación con APIs
    - Almacenamiento local
    - Internacionalización
    - Estilos
    - Acceso a funcionalidades nativas

    <LinkCard title="🚀 APP MicraCity" icon="mdi:rocket" href="/00-Documentacion-MicraCity/dist/app/" />

  </Card>

  <Card title="🚀 Servidor MicraCity" icon="mdi:tools">

    Este proyecto utiliza **Node.js**, **TypeScript** y **Express**, y proporciona:

    - APIs REST para la app móvil
    - Gestión de usuarios y datos
    - Subida y procesamiento de imágenes
    - Conexión con la base de datos MySQL
