import { Card, CardGrid } from '@astrojs/starlight/components';


El proyecto **Micracity** está construido con un conjunto de herramientas modernas del ecosistema **Expo y React Native**.

Estas librerías se utilizan para gestionar:

- navegación
- estado global
- comunicación con APIs
- almacenamiento local
- internacionalización
- estilos
- acceso a funcionalidades nativas

---

## Framework principal

<CardGrid>
  <Card title="React" icon="logos:react">
  
  Librería principal para construir interfaces basadas en **componentes reutilizables**.

  Dependencias:

    - react

  </Card>

  - react-native
  - react-native-web

  </Card>

  <Card title="Expo" icon="logos:expo">

  </CardGrid>
