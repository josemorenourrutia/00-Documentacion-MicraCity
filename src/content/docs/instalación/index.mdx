Esta guía explica cómo crear un proyecto utilizando **Expo y React Native** para desarrollar aplicaciones móviles multiplataforma.

Expo permite desarrollar aplicaciones para:

- Android
- iOS
- Web

utilizando una única base de código.

---

## Requisitos previos

Antes de comenzar, asegúrate de tener instalado:

## Crear un proyecto Expo

Para crear una nueva aplicación ejecuta:

```bash
npx create-expo-app@latest
```

El asistente te pedirá algunas opciones:

What is your app named?
my-app

Which template would you like to use?
Blank (TypeScript)


## Para iniciar el servidor de desarrollo:

```bash
npm run start
```

## Ejecuta el proyecto en un dispositivo Android o emulador.
```bash
npm run android
```
